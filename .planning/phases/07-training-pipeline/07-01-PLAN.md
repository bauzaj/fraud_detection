CRITICAL traps to avoid:
- Label casting: bool('False') == True. Use ONLY df['is_fraud'].map({'True': 1, 'False': 0})
- Feature parity: closed='left' on rolling window excludes current tx, matching processor.py
  pre-append state (history is appended AFTER detect_fraud() checks both features)
- Pipeline wrapping: sklearn Pipeline wraps preprocessor+classifier before joblib.dump so
  Phase 8 calls model.predict_proba(X) without manual encoding

Script structure to implement:

2. Constants (document why they match processor.py values):
   DB_URL = 'postgresql://fraud_user:fraud_pass@localhost:5433/fraud_detection'
   MODEL_PATH = 'src/model/fraud_model.joblib'
   VELOCITY_WINDOW_MINUTES = 5   # matches timedelta(minutes=5) in processor.py
   HISTORY_CAP = 100             # matches processor.py user_transactions cap

                        'hour_of_day', 'day_of_week']

3. Load: pd.read_sql SELECT transaction_id, timestamp, user_id, amount, merchant_category,
   is_fraud FROM transactions ORDER BY timestamp

4. Label casting (TRAIN-02):

   print Class distribution using value_counts()

5. Feature engineering (TRAIN-01) -- sort by user_id then timestamp, set timestamp as index:
   tx_count_last_5min:
     groupby('user_id')['amount'].transform(
         lambda x: x.rolling(f'{VELOCITY_WINDOW_MINUTES}min', closed='left').count()
     ).fillna(0).astype(float)
   amount_vs_user_avg_ratio:
     user_avg = groupby('user_id')['amount'].transform(
         lambda x: x.expanding().mean().shift(1)
     )
     ratio = (df['amount'] / user_avg).fillna(1.0)
   reset_index(), extract hour_of_day (dt.hour) and day_of_week (dt.dayofweek)

6. Build X = df[NUMERIC_FEATURES + CAT_FEATURES], y = df['label']

7. train_test_split test_size=0.2, random_state=42, stratify=y

8. Pipeline (TRAIN-03):

       [('cat', OneHotEncoder(handle_unknown='ignore'), CAT_FEATURES)],
       remainder='passthrough'
   )
   model = Pipeline([
       ('preprocessor', preprocessor),
       ('classifier', RandomForestClassifier(
           n_estimators=100, class_weight='balanced', random_state=42
       )),
   ])

     'WARNING: fraud-class recall X.XXXX < 0.80 target.' otherwise
